CSS制作动画

一、transform的四种变换方式

transform-translate(位移)

transform-rotate(绕某一点旋转)

transform-rotate(绕中轴旋转)

transform-skew(倾斜)

transform-scale(缩放)

二、animation的格式

animation：时长|过渡方式|延迟|次数|方向|填充模式|是否暂停|动画名

声明关键帧

@keyframes xxx{
0%{ }
25%{ }
100%{ }
}
animation的各个属性名 （mdn）
animation-name：动画名
animation-duration：多少s/多少ms 动画时长
animation-timing-function 动画执行的节奏
animation-delay 动画何时开始
animation-iteration-count 动画运行次数
animation-direction 动画是否反向播放
animation-fill-mode 设置CSS动画在执行之前和之后如何将样式应用于其目标。
animation-play-state 运行与暂停
