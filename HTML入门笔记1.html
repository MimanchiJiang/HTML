《HTML入门笔记1》

1.HTML的发明者

HTML的英文全称是 Hyper Text Marked Language，即超文本标记语言。

HTML是由Web的发明者 Tim Berners-Lee和同事zhi Daniel W. Connolly于1990年创立的一种标记语言。

2.HTML起手应该写什么

一般以!DOCTYPE html开头，正常情况下不更改默认配置

3.常用的表章节的标签

  • h1~h6

    • 表示字体大小，从1到6字体依次变小

  • section

    • section表示段落，可用来区分大段文字，一般来说会有包含一个标题。

  • article

    • article表示文章，内部通常内嵌多个section等标签

  • main

    • main通常指主要代码部分

  • aside

    • aside通常指次要代码部分

4.全局属性

  1. class

    2. 给一个标签取个名字

  2. hidden

    3. 让某一个标签消失，不显示在页面中

  3. tabindex

    4. 让你的TAB键能够选择页面中的选项

    tabindex特殊值
    tabindex=0：表示最后选中
    tabindex=-1：表示不要选中它
  4. style

    5. 在style中可写CSS一样的内容（需用“”括起来），同时优先级比CSS高，比JS低

  5. title

    6. 使你的鼠标悬浮在某标签上时，能够显示该标签全部内容。

    通常与

    white-space: nowrap;
overflow: hidden;
text-overflow: ellipsis;
    三句代码同时使用

  6. id

    7. id 全局属性定义了一个全文档唯一的标识符 (ID)。它用于在链接（使用片段）、脚本和样式（通过 CSS）中辨识元素。

    注意！id的名字有诸多限制，不到万不得已不用id！

  7. contenteditable

    8. 使你的任何元素可以被编辑。

5.常用的内容标签

  • a

    • 表示超链接，href内写跳转的网址。

  • strong

    • strong表示内容本身的重要性

  • em

    • em表示语气上的强调

  • code

    • code表示其中的字符以等长的形式表现，通常用于编写代码，英文等

  • pre

    • pre表示保留多个空格或回车等格式
