JS函数

函数是一种对象


setTimeout

let i = 0
for(i = 0; i<6; i++){
  setTimeout(()=>{
    console.log(i)
  },0)
}
打印出的是六个六，而不是12345.
原因是：setTimeout的意思是稍晚一些执行命令，所以等到for循环结束后才会执行console命令。

若需打印出12345，则可将let放入for循环中。
如： 

for(let i = 0; i<6; i++){
  setTimeout(()=>{
    console.log(i)
  },0)
}
这是JS作者对新手逻辑思维的妥协。
