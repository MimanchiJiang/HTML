变换数组

  • let arr=[1,2,3]
  • let arr = new Array(1,2,3)

将字符串变换成数组：

str.split(',')          //','表示用，分隔


Array.from( ,length:)    可以将伪数组变成数组
需满足两个条件：
  • 拥有1，2，3，4下标
  • 有length属性

合并数组

arr1.concat(arr2)

切割数组

arr1.slice(2)          //表示从下标2开始切

arr1.slice(0)          //表示从0开始切，即复制一个数组

删除数组

delete arr[0]          //删除后数组长度不变
修改数组中的length元素会删除元素
删除的正确方式：
  • arr.shift()          //删除数组的第一个

  • arr.pop()          //删除数组的最后一个

  • arr.splice(2,1)          //在2的位置删除1个元素
    arr.splice(2,1,'x','y')          并在删除位置添加x，y

遍历数组

  • for循环：for(let i=0;i<arr.length;i++)
  • arr.forEach(function(item,index) { console.log(`${index}:${item}`) })
差别：for循环可以中断，是个块级作用域；forEach不可以中断，是个函数作用域。

查找某项是否在数组里

arr.indexOf(13)          //返回-1表示没有

查找某项符合要求

arr.find(function(x) {return x%5===0})          //某项是否被5整除，只会返回第一个数
arr.findIndex          //返回对应元素下标

增加数组

  • arr.push(添加的数值)          //在尾部添加
  • arr.unshift(添加的数值)          //在头部添加
  • arr.splice(incex,0,'x')          //在inde处添加

将一个字符串倒转

str.split('').reverse().join('')         //先变成数组，再反转，再合并成字符串

乱序--->顺序

arr.sort()          //默认从小到大

变换数组

数组变换可以让一个数组根据某种自定义规则进行转换
  • map          //n变n
    arr.map(item => item*item)          //数组内的数全部平方

  • filter          //n变少
    arr.filter(item => item%2===0?true:false)        //保留偶数

  • reduce          //n变1
    arr.reduce((sum,item)=>(return{sum+item},0))         //开始的值为0
