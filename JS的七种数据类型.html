类型转换


number --> String

  1. String(n)

  2. n+''或''+n

String --> number

  1. Number(n)

  2. parseInt(n)/parseFloat(n)
  3. n-0

X --> bool

  1. Boolean(x)

  2. !!x

X --> String

  1. String(x)

  2. x.toString()

JS对象


JS对象的两种写法

  • let obj = {'name' : 'xxx','number' : 18}

  • let obj = new object({'name': 'xxx','number':18})

删JS对象

  • delete obj.xxx --> 删除的是属性名

  • delete obj['name'] 

let a ='xxx'
var obj = {[a]:111} --> 表示a是个变量

读JS对象

  • object.keys(obj)可以得到obj的所有key

  • object.values(obj)可以得到obj的所有values

  • object.entries(obj)可以得到obj的所有key和values


修改JS对象

  • 批量赋值：Object.assign(obj,{p1:1,p2:2,p3:3})

  • 修改共有属性：
    • obj.__proto__.xxx(不推荐使用)
    • var person = Object.create(common)

'name' in obj和obj.hasOwnProperty('name') 的区别

'name' in obj 用来检查该属性是否在对象里，包括共有属性


obj.hasOwnProperty('name') 用来检查该属性是否在该对象独有的属性里
